# Joomla 5/6 Skills

Skills for [Joomla 5/6](https://www.joomla.org/) CMS development and customization.

These skills cover modern Joomla development patterns including PSR-4 autoloading, dependency injection, WebAssetManager, and the Helix Ultimate framework.

## Available Skills

| Skill | Description | Install |
|---|---|---|
| [`helix-child-template`](helix-child-template/) | Create and customize child templates with Helix Ultimate | `npx skills add nicolasflores9/skills -s helix-child-template` |
| [`joomla-custom-fields`](joomla-custom-fields/) | Create, manage, and render Custom Fields with FieldsHelper | `npx skills add nicolasflores9/skills -s joomla-custom-fields` |
| [`joomla-database-queries`](joomla-database-queries/) | Database queries: SELECT, INSERT, UPDATE, DELETE, JOINs, Prepared Statements | `npx skills add nicolasflores9/skills -s joomla-database-queries` |
| [`joomla-frontend-integration`](joomla-frontend-integration/) | Frontend integration with WebAssetManager, Bootstrap 5, and responsive design | `npx skills add nicolasflores9/skills -s joomla-frontend-integration` |
| [`joomla-module-development`](joomla-module-development/) | Build custom modules with PSR-4 and modern architecture | `npx skills add nicolasflores9/skills -s joomla-module-development` |
| [`joomla-plugin-development`](joomla-plugin-development/) | Develop plugins using SubscriberInterface and Event Classes | `npx skills add nicolasflores9/skills -s joomla-plugin-development` |
| [`joomla-template-overrides`](joomla-template-overrides/) | Customize output with template overrides and JLayout | `npx skills add nicolasflores9/skills -s joomla-template-overrides` |
| [`sppagebuilder-custom-addon`](sppagebuilder-custom-addon/) | Create custom addons for SP Page Builder v5/6 | `npx skills add nicolasflores9/skills -s sppagebuilder-custom-addon` |

## Quick Install

```bash
# Install all Joomla skills
npx skills add nicolasflores9/skills

# Install a specific skill
npx skills add nicolasflores9/skills -s joomla-module-development
```

## Requirements

- Joomla 5.x or 6.x
- PHP 8.2+
- MySQL 8.0+ or PostgreSQL 15+
